# トラブルシューティングログ: ローカル起動時のViteコマンド未検出

- **発生日時**: 2026-01-27 13:48
- **事象**: `npm run dev` を実行した際、`'vite' は、内部コマンドまたは外部コマンド...として認識されていません` (exit code 1) というエラーが発生し、起動に失敗した。
- **エラー内容**: `node_modules` フォルダが存在せず、依存関係がインストールされていないことが判明。

## 原因
Vercel移行作業中の `npm install` が正常に完了していなかった、あるいは実行待機中にプロセスが終了・中断していた可能性がある。
エージェントがビルド成功を仮定して進行してしまったため、実際には依存パッケージが欠落していた。

## 解決策
**依存関係の再インストールを行う。**

### 手順
1. `node_modules` の不在を確認。
2. 以下のコマンドを実行し、完了を確実に待機する。
   ```bash
   npm install
   ```
3. `node_modules/vite` の存在を確認後、再度 `npm run dev` を実行する。

## 今後の対策
`npm install` などの長時間要するコマンドを実行した際は、必ず終了ステータスと生成物（`node_modules`）の存在を物理的に確認してから次のステップ（ビルドや起動）へ進むこと。
