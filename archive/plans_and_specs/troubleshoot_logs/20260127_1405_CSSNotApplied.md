# トラブルシューティングログ: CSSが適用されない

- **発生日時**: 2026-01-27 14:05
- **事象**: アプリケーションは起動するが、スタイルが崩れており、CSSが適用されていない状態。
- **エラー内容**: ブラウザのネットワークタブを確認するとCSSファイルが読み込まれていない。

## 原因
Viteプロジェクトへの移行時に `style.css` を `src/style.css` に配置したが、エントリーポイントである `src/main.js` または `index.html` からの読み込み記述（インポート）が漏れていたため。
Viteでは、JSファイル側でCSSをインポートすることでビルドプロセスに含まれる仕組みだが、それが実装されていなかった。

## 解決策
**`src/main.js` にCSSのインポート文を追加する。**

### 手順
1. `src/main.js` の先頭（またはデータインポート付近）に以下を追記する。
   ```javascript
   import './style.css';
   ```
2. 保存後、HMR（ホットモジュールリプレースメント）により即座にスタイルが反映されることを確認する。

## 今後の対策
ファイルやフォルダを移行した際は、単に配置するだけでなく、エントリーポイントからの「参照」が維持・更新されているかを必ずセットで確認する。
特にVite等のバンドルツールを使用する場合、HTMLでの `<link>` ではなく JS import が必要になるケースが多い点に留意する。
