# Semantics and Terminology

プロジェクト・UI・ロジック内部で使用される語彙の定義です。

## アプリ固有の概念
- **サイクル (Cycle Time / Average Cycle)**
  1回のキャスティングアクション全体にかかる時間。キャスト → 待ち時間 → ルアーアクション（あれば） → フッキング・釣り上げ → 再キャスト可能な待機状態、までの一連の平均合計時間（秒）。
- **シナリオ (Scenario)**
  使用ルアーの「発見」「型確定」状態をステップごとに指定した一意の行動経路。（例: `2回アクション/1回目発見/2回目型確定`）
- **期待時間 (Expected Time)**
  ターゲット魚を「1匹」釣るために必要となる時間（秒）の期待値。全体のサイクル時間をターゲットヒット率で割って算出される。
- **基本重み / 最終重み (Base Weight / Final Weight) / ヒット率 (Hit Rate)**
  ゲーム内のヒット判定ロジックをシミュレートするための数値。総重みに対する自身の最終重みの割合が、その魚が掛かる「ヒット率（確率）」になる。
- **GDS (Game Domain Settings)**
  `calculator.js` 内で定義されているシステム定数群（キャスト時間 `D_CAST`、ルアーアクション時間 `D_LURE`、等）。
- **変数モード (Variable Mode)**
  未知の重みを持つ魚の基本重みを $p$（パーセンテージ）として扱い、それ以外の係数成分を抽出することで確率変化による影響度を一覧化する計算モード。
